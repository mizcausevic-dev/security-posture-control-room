# Security Policy

## Scope

This repository is a portfolio artifact and internal-tool simulation. It does not include production secrets, live customer data, or real policy records.

## Reporting

If you identify a security issue in the code, documentation, or CI setup, please report it privately through the public portfolio channels linked in the README.

## Data Notes

- all compliance records are fictional
- no credentials should ever be committed
- `.env.example` is provided only for local configuration patterns
